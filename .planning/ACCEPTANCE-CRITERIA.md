# Acceptance Criteria

**Updated:** 2026-03-06
**Source:** Clean slate rewrite for v5.0 milestone

## Milestone-Level Criteria

- [ ] Planning documents for a 50-phase project contain only the active milestone's phases
- [ ] Agent prompts work as a coordinated system -- no isolated prompt islands
- [ ] Spec drift between `.planning/` and codebase is detectable via a single command
- [ ] Init flows produce context thorough enough that fresh agents never need clarifying questions
- [ ] Every workflow step has a corresponding skill -- no gaps in coverage
- [ ] `npm run build` and `npm test` pass after every phase

## Phase-Level Criteria

Populated per-phase during `/maxsim:discuss-phase` and `/maxsim:plan-phase`.
